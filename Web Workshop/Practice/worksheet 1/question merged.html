Q1-What is HTML? Write a short defination and list three features of HTML in your own words

Ans-HTML(HyperText Markup Language)is the standard language used to create and structure web pages on the internet. It uses tags to define elements like text,images,links,and forms.

Three Key Features of HTML:

  • 1.Easy to Learn-Uses simple tags and syntax.
  • 2.Platform Independent-Works on all browsers and operating systems.
  • 3.Supports Multimedia-Allows images,audio,video,and links.
